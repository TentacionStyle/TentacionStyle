### 🏠 HomeLab & Self-Hosting
Me apasiona el **self-hosting** y la soberanía de datos. Mantengo mi propia infraestructura doméstica basada en **Docker**:

* **🎬 Jellyfin:** Centro multimedia (Mi "Netflix" privado).

* **🌐 Nginx Proxy Manager:** Gestión de certificados SSL y Reverse Proxy.
* **🚀 Tailscale:** Red mesh privada para acceso seguro remoto.
